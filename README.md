# Online Shoe Store

## Project Description

This is a simple React application of an online shoe store. Users can view different shoes, add them to the shopping cart, increase or decrease the quantity of items, and see the total amount of their cart.

The project is created using React and the useState hook for state management.

---

## Features

- Display different shoes with image, name and price.
- Add shoes to the shopping cart.
- Increase or decrease the quantity using + and - buttons.
- Automatically remove the item when quantity becomes zero.
- Display the total price of all items in the cart.
- Simple and responsive user interface.

---

## Installation

Clone the repository or download the project.

Install all dependencies.

npm install

## Run the Project

Start the development server.

npm run dev

Open the browser and visit the local URL shown in the terminal (usually http://localhost:5173).

## Build for Production

npm run build
